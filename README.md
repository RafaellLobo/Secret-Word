## 🇧🇷 Sobre o Projeto

Este projeto foi desenvolvido como parte prática do curso **"React do Zero a Maestria"**, ministrado por Matheus Battisti (Hora de Codar).

A aplicação consiste em um jogo interativo construído com React, onde foram aplicados conceitos fundamentais e avançados da biblioteca, incluindo gerenciamento de estado, lógica de jogo e componentização.

---

## 🇺🇸 About the Project

This project was developed as a practical application from the course **"React from Zero to Mastery"**, taught by Matheus Battisti.

It is an interactive game built with React, applying both fundamental and advanced concepts such as state management, game logic, and component-based architecture.
