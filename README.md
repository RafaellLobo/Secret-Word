## 🇧🇷 Sobre o Projeto

O **Secret Word** é um jogo interativo desenvolvido em React como parte prática do curso **"React do Zero a Maestria"**, ministrado por Matheus Battisti (Hora de Codar).

O objetivo do jogo é adivinhar palavras com base em categorias, utilizando lógica de tentativas e pontuação. O projeto aplica conceitos essenciais e avançados do React, com foco em componentização e gerenciamento de estado.

---

## 🇺🇸 About the Project

**Secret Word** is an interactive game built with React as part of the course **"React from Zero to Mastery"**, taught by Matheus Battisti.

The goal is to guess words based on categories, using a system of attempts and scoring. This project applies both fundamental and advanced React concepts, focusing on component-based architecture and state management.
